# Adding new lines
- Fork the repository
- Update the file in assets/data/ or create a new one
  - For voice lines, please use the actual translation made by the various versions
  - You can take as example on that is already made, please write the character name without any space for the filename
  - For the Japanese lines, please use [Japanese punctuation](https://en.wikipedia.org/wiki/Japanese_punctuation)
- Make a pull request
